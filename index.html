PROJECT ABOUT CONSOLE

Visual Studio is a coding platform. There are a few steps to setting it up. First, download Visual Studio. Then, open up Visual Studio and go into the extensions section. There download Live Server. Then, click File. Next click Open. Then, click on the folder you want to put your project in. Then, add the files you want to add and you're finished.

Console is something in which you can figure out errors in your code and see text you put in there from your code. How you find the console is you go to your webpage, right click, click on inspect and click on Console.
